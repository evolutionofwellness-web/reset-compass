About The Reset Compass

The Reset Compass helps you quickly choose a short, intentional activity to reset your focus, energy, or mood. Directions represent different wellness modes; each mode suggests short "Activities" you can do immediately.

How to use

  1. From the compass: either tap a direction or pick a Mode Card below.
  2. In the Mode modal, select an Activity (it highlights) â€” selecting is separate from "Begin Activity".
  3. Tap "Begin Activity" to perform and record the activity. That increments your daily streak (if applicable).
  4. Use the "Quick Wins" button in the header for a curated list of fast actions (outside the compass).

The current mode names used by this build are: Surviving, Drifting, Grounded, Growing. Your theme (light/dark) preference and streak are saved locally. Use the Home / Return links to get back to the compass.
