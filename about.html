About The Reset Compass

The Reset Compass helps you quickly choose a short, intentional activity to reset your focus, energy, or mood. Wedges represent different wellness modes. Each mode suggests short activities you can do immediately.

How to use

  1. Tap a wedge inside the compass or pick a Mode Card below.
  2. In the Mode dialog, select one or more Activities (they highlight). Use the per-activity Notes inputs if you want.
  3. Add optional notes about each selected activity before you Complete them.
  4. Tap "Complete Selected" to record the selected activities. That updates your daily streak.
  5. Quick Wins are available using the Quick Wins button in the header for very fast, one-step resets.

The mode names used by this build are: Growing, Grounded, Drifting, Surviving. Your theme (light/dark), streak, and history are saved locally.
